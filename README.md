# FPU
Float Pointing Unit (mathematical processor). For connecting to the CPU_8080 processor board - https://github.com/R2AKT/CPU_8080.

Note: The processor module CPU_8080 must use a 580VK38 (8238) system bus controller, which is not functional when using 580VK28 (8228).

Status - in the queue for the payment order.

Модуль математического процессора. Для подключения к процессорной плате CPU_8080 - https://github.com/R2AKT/CPU_8080.

Примечание: в процессорном модуле CPU_8080 должен применяться контроллер системной шины 580ВК38 (8238), при использовании 580ВК28 (8228) не работоспособно.

Статус - в очереди на заказ платы.
